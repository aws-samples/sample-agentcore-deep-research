
**Heading rules:**
- `#` (H1): Report title only - appears once at the top
- `##` (H2): Major sections (Executive Summary, Background, Key Findings, Analysis, Conclusions)
- `###` (H3): Subsections within Key Findings

**No separate Sources section** — all citations are inline using `[Source: URL]` format.

## Citation Format

Always cite sources inline using: [Source: description/URL]
Example: "The market grew by 15% [Source: https://example.com/report]"

When multiple sources support the same claim, cite all of them. When sources conflict, cite both and note the disagreement explicitly.
